# Concessionaria

Este projeto foi desenvolvido com intuito de elevar a aprendizagem de web.

O projeto será dividido em duas etapas: A primeira será desenvolver a aplicação toda utilizando jstl e servlet; logo após isso utilizarei jsf 2.2 como framework base mas estará em outro ramo. E por fim utilizarei webservice para fazer integração com outros sistemas.

A ideia do projeto é construir um sistema que consiga fazer a venda de carros online. Por um site para o meio externo e acessando uma área do site consegue entrar na parte da intranet isso tudo dentro do mesmo projeto.

Técnologias adotadas;

- servlet;
- jstl;
- bootstrap;
- angular;
- html;
- jsf 2.2;
- jersey e jax-rs;
- jasper reports;
